## Overview
Excel-based **Expense Tracking Dashboard** to monitor and analyze site expenses over time.  
**Note:** All data is **mock/demo data**.

---

## Features
- Record expenses by Date, Site, Type, Amount, Notes  
- Data validation with Site and Expense Type lists  
- Automatic Month & Year extraction via Power Query  
- Pivot Tables for totals, monthly trends, and yearly summaries  
- Interactive dashboard with charts, slicers, and KPI metrics  

---

## Tools
- Microsoft Excel (Tables, Pivot Tables, Charts, Slicers)  
- Power Query  

---

## Usage
1. Update `Raw_Data.xlsx` with expenses  
2. Refresh Power Query and Pivot Tables  
3. Explore trends using dashboard slicers
